# Start Slidev Development Server

Starts the Slidev development server for live preview of slides.

## Steps

1. Navigate to the `slidev/` directory
2. Run `npm run dev` (or `pnpm dev` if using pnpm)
3. The server will open automatically in your browser at `http://localhost:3030`

## Usage

Run this command when you want to:
- Preview slides in real-time while editing
- Test interactive components and animations
- Verify slide layout and styling

## Notes

- The dev server watches for file changes and hot-reloads automatically
- Press `Ctrl+C` in the terminal to stop the server
- Make sure you're in the `slidev/` directory before running
