# Export Slidev to PDF/PNG

Exports the Slidev presentation to PDF or PNG format.

## Steps

1. Navigate to the `slidev/` directory
2. Run `npm run export` (or `pnpm export`)
3. Follow the prompts to choose export format (PDF or PNG)

## Usage

Run this command when you want to:
- Generate a PDF handout for participants
- Create PNG screenshots of individual slides
- Export for offline viewing or printing

## Notes

- PDF export requires the dev server to be running or will start it automatically
- Export quality can be configured in Slidev settings
- Large presentations may take time to export
