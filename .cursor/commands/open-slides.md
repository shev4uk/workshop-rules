# Open Main Slides File

Opens the main slide deck file for editing.

## Steps

1. Open `slidev/slides.md` in the editor
2. This is the primary file containing all slide content

## Usage

Run this command when you want to:
- Edit slide content quickly
- Review the main presentation structure
- Add or modify slides

## Notes

- Main slides are in `slidev/slides.md`
- Additional slide pages can be imported using `src:` attribute
- See `slidev/pages/` for imported slide files
